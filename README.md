# ShowHost

A simple web-app that serves a single purpose: Display the hostname to the user.
Dockerized, to make it easy to deploy the project anywhere where you always wanted to know the hostname from.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

